### How to Set Up

1. Copy and paste into the terminal

```
git clone https://github.com/nliinnn/course-organizer.git
```

2. After the repository has been cloned, cd into the new folder

```
cd my-react-app
```

3. Install the dependencies

```
npm install
```

4. Run the project

```
npm start
```
